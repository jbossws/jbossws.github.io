JBossWS 4.0.1 release

By JBossWS Team | February 5, 2012

JBoss AS 7.1.0.Final is coming soon, so here we are with the last minute updates in the webservices area with another JBossWS release ;-) JBossWS 4.0.1 includes important bug fixes and covers some application server integration changes for improving management. Besides that Apache CXF has been upgraded to the 2.4.6 release which also comes with many improvements and bug fixes.

Have a look at the release notes for further details [1][2]! Feedback is welcome as always.
