Here below is the list of the released JBossWS 3.x versions; you'll find three tables, one for each JBossWS stack.

The binary distribution downloads include the selected stack and integration code libraries, the installation scripts, documentation and the common testsuite sources. The source distribution downloads, instead, include all the sources (including those of the full stack specific testsuites).

Please verify the supported target containers tables when selecting the JBossWS version to download.
