JBossWS-CXF 7.3.7.Final has been released. In this bugfix release we added JDK25 support, fixed expired certificates issue and changed bus
default configuration to avoid accumulated cleanup threads leak.
For complete issue list, please check the [Release Notes](https://issues.redhat.com/secure/ReleaseNote.jspa?projectId=12310050&version=12471164).

Please try out this release, and let us know if you have any issues or questions.
