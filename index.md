JBossWS-CXF 7.3.8.Final has been released. In this bugfix release we added a functionality that lets JBossWS CXF 
accept encoded URL paths, e.g. those including NLS (National Language Symbols) characters in Web Services names.

See more details in [the last Blog post]({% post_url 2026-05-26-jbossws-738-release %})

Please try out this release, and let us know if you have any issues or questions.
